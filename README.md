# Software Architect Portfolio

A modern, responsive personal portfolio website designed for software architects and senior technical professionals.

## Features

- **Modern Design**: Clean, professional UI with smooth animations and transitions
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Elements**: Scroll animations, hover effects, and dynamic content
- **Sections Include**:
  - Hero section with call-to-action
  - About section with statistics
  - Technical expertise showcase
  - Professional experience timeline
  - Featured projects with metrics
  - Contact form and social links

## Technologies Used

- HTML5
- CSS3 (with CSS Variables and Grid/Flexbox)
- Vanilla JavaScript
- Google Fonts (Inter)

## Customization

### Personal Information

1. **Hero Section** (`index.html` lines 32-54):
   - Update the title and descriptions
   - Modify call-to-action buttons

2. **About Section** (`index.html` lines 56-90):
   - Add your personal story
   - Update statistics (years of experience, projects, etc.)

3. **Skills Section** (`index.html` lines 92-170):
   - Customize the skill categories
   - Add or remove technologies based on your expertise

4. **Experience Section** (`index.html` lines 172-232):
   - Add your work history
   - Update job titles, companies, and descriptions
   - Modify the timeline dates

5. **Projects Section** (`index.html` lines 234-316):
   - Showcase your best projects
   - Update project descriptions and metrics
   - Add relevant technologies used

6. **Contact Section** (`index.html` lines 318-360):
   - Update email address
   - Add your social media profiles (LinkedIn, GitHub, Twitter)
   - Configure form submission endpoint

### Styling

All styling is contained in `styles.css`. You can customize:

- **Color Scheme** (lines 7-20): Update CSS variables for primary colors, gradients, etc.
- **Typography**: Modify font sizes and families
- **Spacing**: Adjust padding and margins
- **Animations**: Customize transition speeds and effects

### Branding

1. **Logo** (`index.html` line 17): Replace `<SA/>` with your initials or logo
2. **Title & Meta**: Update the page title and meta description for SEO
3. **Favicon**: Add your favicon in the `<head>` section

## Setup

1. Clone or download this repository
2. Open `index.html` in a web browser
3. Customize the content to match your profile

## Deployment

This is a static website and can be deployed to:

- **GitHub Pages**: Push to a repository and enable GitHub Pages
- **Netlify**: Drag and drop the folder for instant deployment
- **Vercel**: Connect your repository for automatic deployments
- **AWS S3**: Host as a static website
- **Any web server**: Upload files via FTP
